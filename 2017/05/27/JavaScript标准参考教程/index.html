导论部分：

导论部分介绍了JS的发展历史，以及目前的发展近况，算是长姿势部分，读起来也是蛮有意思的，也有阮老师一些让人耳目一新的个人见解，推荐阅读，可以让你慢慢进入这本书。

知识点记录：

Java Script和Java没有直接关系，它是一种嵌入式语言（embedded），其本身不提供任何与I/O（输入/输出）相关的API，都要依靠宿主环境（host）提供，只适合嵌入大型应用程序环境，调用宿主环境提供的底层API。
宿主环境：浏览器，服务器（Node项目）。
JS：对象模型语言，宿主环境通过这个模型，描述自己的功能和操作接口，通过JS控制这些功能。
浏览器API:

- 浏览器控制类：操作浏览器
- DOM：Document Object Model
- BOM: borwser Object Model

服务器API(Node)：

- 操作系统API
- 文件操作API
- 网络通信API

JS: 支持面向对象、面向过程、函数式编程，很灵活，各种写法。
1990年 万维网–>命令行操作，字符窗口
1992年 NCSA–>Mosaic,图形界面窗口
1994年 Netscape–>Netscape Navigator
Netscape（网景公司）因最初的互联网条件及浏览器环境，迫切需要一种嵌入式脚本语言，控制浏览器，来实现简单的表单验证。
JS的基本语法有C和Java的影子，数据结构中原始值、对象等借鉴自Java，函数是与其他数据类型可以平起平坐的，所以可以使用函数式编程，在这里也引入了一个闭包的概念。在面向对象的的编程中使用了原型继承模型。

Mocha–>LiveScript–>JavaScript–>ECMAScript
-这里并不是表示JS变成了ES，而是说JS促成了ES，在《JS高级程序设计》中有专门一节介绍了JS的实现，ES是作为其核心内容和基础。
ES只是JS语言基本语法结构的标准化。
目前ES已结更新到ES7，按照以后一年一个版本的说法，只能说一句：路漫漫其修远兮，吾将上下而求索。

  • 以前一直以为第一个浏览器是Netscape，原来还有NCSA这会事。突然就联想到美国控制着全球网络这个事了。

语法部分：

这部分可以回顾下JS的基本语法、数据类型、对象、函数等基础知识。此外书中还特别介绍了原生JS错误处理机制，以及编程风格。虽然只有短短两节，但是也可以在脑子中留下印象。虽然这两部分其实早已融入平时编码中。还是推荐再看一下，特别是回顾下自己的代码风格。把编码当做自己的作品，用心去对待。

知识点记录：

line：JS逐行执行，执行单位为行（line）。
statement：执行语句。
expression：表达式，一定是要返回一个值。
空语句：; 可以作为JS安全语句。
Variable: 变量，使用变量等同于引用值。JS的声明和赋值在编码时可以写在同一行，但在JS引擎解析下，还是会先声明变量，这也是之后变量提升的由来，建议书写中声明部分之前提前。JS中的变量可以只声明不赋值，这个变量将会引用一个undefined的数据类型，作为自己的值保存起来。

- 变量会根据所赋值改变数据类型。
- 已经声明的变量在同一作用域（scope）下再次声明是无效的。
- 已经声明的变量在同一作用域（scope）下再次声明且赋值，其实执行的就是变量赋值。

delete: 无法删除顶级的window属性，无法删除声明了的变量。
Variable Hoisting：变量提升，在JS引擎解析下，所有被声明的变量都很先提升到当前作用域顶部。赋值部分位置不变。
Identifier：标识符–>用来识别对象的名称，变量名，函数名都是常见的标识符。JS中的标识符有命名规则，这里就不再赘述。
block：区块，JS使用的大括号，{}，大括号会将其内部的相关语句组合在一起。
scope：作用域，全局作用域及函数级作用域。
条件语句：

  • else总是跟随距离自己最近的if语句。
  • switch语句无break，贯穿现象，switch内部使用严格相对运算符（===）。
  • 三元运算符：(condition) ? expr1:expr2

break: 跳出代码块或循环。
continue: 跳过当次（本轮）的循环。
对于多重嵌套循环，break与condition只针对最内层（本层及子层）
label：定位符（可以是任意标识符），用于跳转至程序任意位置。
数据类型：按照《JavaScript高级程序设计》理解的是 基本数据类型和引用数据类型，在这本书中阮老师使用原始数据类型、合成类型和其他来区分。
Primitive type: 原始类型（number,string,boolean）
Complex type: 合成类型（object）
特殊值： undefined、null
Object：狭义的对象仅指Object，array，function都是一种对象类型，要是再往广义说，那就是万物皆对象了。
function：处理数据的方法,JS把函数当成一种数据类型，typeof(function)返回function，可以赋值、传递。
typeof(): 原始数据类型、undefined及function返回自己本身类型，其他都是Object，包括window,[],{}。
undefined： 以下四种情况返回undefined

- 变量声明未赋值
- 函数没有返回值
- 对象没有赋值的属性
- 调用函数时不提供应有的参数

Float：浮点数存在不精确问题。
NaN: 不等于任何数包括自己。
isNaN(): 只对数值有效，传入其他值会先被转成数值。
parseInt(): 字符串转整数，可以传入第二个参数用来做进制转换
parseFloat(): 字符串转浮点数。
object：对象，万恶的核心，无序数据集合，键值对的形式。
键值对：键名–>property（属性），键值为函数时，代表方法。
点运算符与方括号运算符都可以用来读取属性，但是数组中不用点运算符。
全局变量都是window的属性。
Object.keys(obj)：查看对象的所有属性。
delete：删除对象本身的属性，无法删除window属性，也无法删除继承的属性。
in运算符：检查某个属性是否在对象内，不能识别继承的属性。
for…in: 用来遍历一个对象的全部属性，只遍历(enumerable)可以遍历到的属性，跳过不开遍历的。包括自身属性及所有继承的属性。
Array：多维数组含义，数组可以看成一个键名为数值键且从’0’开始特殊对象。
arr.length=0: 数组清空。
array-like object： 类似数组对象，只要有length属性就是类似数组的对象，arguments，DOM元素集，字符串。
var arr=Array.prototype.slice.call(array-like): 类数组对象转数组
空位元素
Function Declaration:函数声明
Function Expression：函数表达式
构造函数：只有最后一个函数会被当做函数体
函数重复声明覆盖
函数声明提升
函数的属性：

- name：function之后的函数名
- length：定义函数时的形式参数个数，调用时不改变
- toString(): 返回函数源码包括内部注释

函数级作用域：function内局部变量；只作用在自身内对内部函数不起作用；函数执行时所在的作用域是定义时的作用域。
pass by value：对于原始数据类型使用传值传递，函数内部修改不影响原始值
pass by reference：对于复合数据类型使用传址传递，函数内部修改属性值会修改原始属性。
chain scope：链式作用域
closure：闭包

- 实现在函数外部读取函数内部声明的变量
- chain scope 子对象会向上逐级寻找父对象的变量

运用chain scope，把子对象作为父对象返回值实现外部读取内部变量。

- 定义在一个函数内部的函数
- 特点：记住诞生环境，得到诞生环境内部变量
- 作用：读取函数内部变量；让这些变量始终保持在内存中；函数内部作用域接口；封装对象的私有属性和私有方法。
- 正因为闭包会保存在内存中，所以不能滥用。多了会影响网页性能。

IIFE：理解调用的函数表达式（匿名函数使用）

- 圆括号方法
- 运算符方法
- new方法
- true&&方法
- 0;方法
- 目的：不必为函数命名，避免污染全局变量；内部形成单独作用域，可封装一些无法读取的私有变量

Error：JS原生错误类型

- SyntaxError：语法错误
- ReferenceError：引用一个不存在的变量
- RangeError：超出有效值范围
- TypeError：变量或参数不是预期类型
- URIError：URI相关函数参数不正确

Object对象：

Object:对象；原生JS提供Object对象，所有其他对象都继承自这个对象。Object本身也是一个构造函数，可以直接通过它来生成新对象。

知识点记录：

字面量写法：var obj={}，与new Object()等价。
prototype属性：所有构造函数都有一个prototype属性，指向一个原型对象。
Object.keys(obj)：返回一个数组，包含所有可枚举的属性。
Object。getOwnPropertyNames(obj):返回一个数组，包含所有属性，包括不可枚举的属性，如.length。
二者返回的都是本身的属性，不是继承的。

对象属性模型的相关方法:

- Object.getOwnPropertyDescriptor(): 获取某个属性的attributes对象；
- Object.defineProperty(): 通过attributes对象，定义某个属性；
- Object.defineProperties(): 通过attributes对象，定义多个属性；
- Object.getOwnPropertyNames(): 返回直接定义在某个对象上面的全部属性名称。

控制对象状态的方法：

- Object.preventExtensions(): 防止对象扩展；
- Object.isExtensible(): 判断对象是否可扩展；
- Object.seal(): 禁止对象配置；
- Object.isSealed(): 判断一个对象是否可配置；
- Object.freeze(): 冻结一个对象；
- Object.isFrozen(): 判断一个对象是否被冻结。

原型链相关方法：

- Object.create(): 该方法可以指定原型对象和属性，返回一个新的对象；
- Object.getPrototypeOf(): 获取对象的Prototype对象
